### 一、win10常用快捷键

```
ctrl+tab			切换浏览器标签页
ctrl+w 				关闭浏览器当前页
ctrl+shift+t 		恢复关闭的浏览器页面
ctrl+t 				打开新标签页
ctrl++				放大页面
ctrl+-				缩小页面
ctrl+alt+delete 	打开任务管理器
ctrl+p				打印
ctrl+win+d 			新建桌面



alt+tab 			切换任务窗口 
alt+d 				焦点固定到地址栏
alt+enter			查看选中文件属性
alt+前进/后退方向键   浏览器页面后退前进

win+D 				回到桌面，再按一次，回复原来的窗口
win+L 				锁定桌面
win+R 				打开运行
win+E				打开资源管理器
win+m				最小化窗口
win+方向键，		 可移动窗口
WIN+数字键：		 切换任务栏的窗口，没有窗口的会打开新窗口。
win+t 				循环切换任务栏
win+x 				打开“Windows移动中心”设置面板
win+Q               打开搜索框

F1 					显示当前程序或者windows的帮助内容
F2				    如果选中文件的话，进行重命名
F3					查找
F5					浏览器页面刷新
F6					使用浏览器时，地址栏获得焦点（即光标移到了地址栏）
F11					浏览器全屏
F12					浏览器审查元素/调试界面
Home 				回到当前行首
end 				回到当前行尾
prtsc 				截屏
```

### 二、UltraEdit常见快捷键

```
F7				插入当前日期/时间 
CTRL + W		自动换行 
CTRL + T		段落重新格式化
CTRL + F5		转换所选文字为小写 
Alt + F5	 	转换所选文字为大写 
CTRL+E 			删除当前行内容

shift + 鼠标左键双击  相同字符串高亮
```

### 三、Navicat 快捷键

```
Ctrl+Q            打开查询窗口
Ctrl+W            关闭一个查询窗口

Ctrl+/            注释sql语句
Ctrl+Shift +/     解除注释

Ctrl+R            运行查询窗口的sql语句
Ctrl+Shift+R      只运行选中的sql语句
 
Ctrl+L            删除一行 (复制Ctrl +D) 

Ctrl+D/O          表的数据显示显示页面切换到表的结构设计页面 
```
